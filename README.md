# Chatroom code demo starts

## demo

[experience it](http://socket.vjscoder.com/websocket-chatroom/index.html#/)

## start local server

```bash
# Open the command line under the src/server folder

node index.js

# or npm i -g nodemon，then：
nodemon index.js
```

## Start the front-end code

``` bash
npm install
```

## Start

```bash
npm run serve
```

## Build

```bash
npm run build
```
